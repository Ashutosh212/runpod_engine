About this tool

Upload any 3-band RGB satellite image to detect vehicles using an oriented bounding box model.
After inference, download the raw prediction file in DOTA format — compatible with standard accuracy evaluation pipelines.

✓Formats  PNG · JPG · TIFF (3-band RGB only)
✓Max size  2 GB  (≈ 30 km² at 0.3 m/px)
✓Large images  auto-tiled into 1280×720 patches
✓Output  DOTA .txt — use directly for mAP / accuracy evaluation
